Welcome to Play-Angular Seed play-angular-require-seed

A web development application template to start developing your reactive Single Page Web Application. The stack consists of Playframework 2.4, WebJars, RequireJS and AngularJS.

When you are done developing, run sbt dist on the console to get an optimized, production-ready app. To try out production mode, execute sbt start.

Enjoy!
